## :information_source: How To Use

1. Clone repo

```bash
git clone https://github.com/UrielGui/parking.git
```

2. Install Dependencies

```bash
yarn install
```

3. Run the app

```bash
yarn start
Open http://localhost:3000 to view it in the browser.
```
